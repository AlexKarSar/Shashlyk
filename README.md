# Shashlyk
Мат. модель процесса приготовления шашлыка
Задача: Требуется смоделировать процесс приготовления шашлыка. 

Шашлык жарится на мангале (угли обогревают часть шашлыка) и под воздействием температуры физические свойства куска постепенно меняются. 
Можно считать, что кусок мяса имеет форму цилиндра и прогревается сбоку равномерно. 
Шашлык готов, когда температура в центре (заданном радиусе) куска в течении некоторого времени остается не ниже определенного (заранее известного) значения.
